# ЕКДЭ — экзамен 2026

Чёрно-белый сайт-экзамен «Е-К-Д-Э» на React + TypeScript + Vite + Tailwind CSS.
Полностью клиентское приложение: все данные хранятся в браузере (localStorage,
IndexedDB, Cache API, cookie), бэкенд не требуется.

## Сценарий

1. Экран приветствия с правилами и переключателем темы (чёрная / белая).
2. Регистрация — пользователь вводит никнейм (сохраняется навсегда).
3. Экзамен — 16 вопросов, таймер на 25 минут.
4. Свернуть/закрыть вкладку во время экзамена → попытка аннулируется
   и пользователь больше не сможет начать заново.
5. По завершении — экран результатов с никнеймом и баллами.
6. Отдельная вкладка «Проверка ответов» показывает, на какие вопросы
   ответ был правильным, без раскрытия самих ответов.

## Подсчёт баллов

- 15 основных вопросов × 6 баллов = **90 баллов максимум**.
- 16-й вопрос — бонусный (0 баллов).

## Сохранение состояния

Состояние экзамена дублируется в нескольких хранилищах сразу, чтобы простая
очистка cookies или одного из хранилищ не вернула возможность пройти экзамен:

- `localStorage` (с несколькими резервными ключами),
- `sessionStorage`,
- cookies,
- IndexedDB,
- Cache API.

При загрузке состояние «аннулировано» или «завершено» из любого хранилища
имеет приоритет над «свежим» состоянием.

> Полностью защититься от очистки данных сайта в браузере на чистом фронтенде
> невозможно — это потребовало бы бэкенда. Текущее решение делает обход
> достаточно неудобным для обычного пользователя.

## Запуск

```bash
npm install
npm run dev      # запуск dev-сервера на http://localhost:5173
npm run build    # сборка production
npm run preview  # предпросмотр production-сборки
npm run lint     # проверка ESLint
```

## Стек

- React 19, TypeScript 6, Vite 8
- React Router v6 (HashRouter — статический хостинг работает без настройки)
- Tailwind CSS 3 с включённым `darkMode: "class"`
- ESLint + typescript-eslint

## Структура

```
src/
├── components/      # Timer, ThemeToggle
├── context/         # ExamProvider + useExam
├── data/            # вопросы и подсчёт баллов
├── lib/             # многослойное сохранение, темы
├── pages/           # Welcome, Register, Exam, Results, Review, Annulled
├── App.tsx          # маршрутизация
└── types.ts         # доменные типы
```
